Wormhole Redux

Wormhole Redux is a fast paced multiplayer arcade game of survival. Pilot ships in a race to collect powerups and fire them into enemy wormholes. The last man (or team) standing wins.

Join the discussion!

About

Wormhole was a game that was originally released in the early 2000s by Centerfleet (later Centerscore). The game also showed up on Newgrounds after becoming popular. Sadly, on December 31 2002, the game and website were shutdown.

Wormhole Redux is a project that seeks to revive classic Wormhole. Currently, there are no permanent accounts, no clans, no rankings, etc. However, the classic gameplay is there!

If you are new to Wormhole, the original guide has a good explanation of the interface and how to play.

Download

Wormhole Redux is a java application. You will need Java 1.6 or higher (JRE), which is likely to be already installed.

Wormhole Redux Client EXE file

The EXE file will run on Windows PCs.

Wormhole Redux Client JAR file

The JAR file will run on Linux and Mac (and Windows) PCs. With Java installed, you can usually double click a JAR, just like an exe file, and it will open. Otherwise, you will need to run something like java -jar whclient.jar from the terminal (command line), after navigating to the directory where the JAR resides.

FAQ

Is the game free?
Yes, it is free. There is no cost and there will never be a cost. No donations are accepted.

No one is online?
There are not a ton of players at the moment, and people don't seem to idle/hangout in game. Best way to setup some games is to get on discord. Also, just to test it out yourself, you can boot up multiple clients.

Why is the game window so small?
It used to be even smaller; the game was java applet that ran in the browser at ~800x550px. This seems small now with modern monitor sizes and screen resolutions. We increased the game window size to a reasonable middle-ground (~1150x750px), but it cannot be resized.

What are the future plans?
Not sure yet. Minor updates or bug fixes will continue to be made for some time, but major changes will depend on the interest and community support.

How were you able to recreate the original Wormhole?
The client was released on the internet several years ago. There were some issues that had to be fixed, but we were able to piece together a server by analyzing the client.
